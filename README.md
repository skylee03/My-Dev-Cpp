## Main Features

- [MinGW-W64](http://www.mingw-w64.org/) GCC 8.1.0 32/64bit
- Syntax highlighting
- Code completion
- Code insight
- Editable shortcuts
- GPROF profiling
- GDB debugging
- [AStyle](http://astyle.sourceforge.net/) code formatting
- Devpak IDE extensions
- External tools
